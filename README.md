# damn
--------------------------
## 这是一个无聊的程序
## 可以输出一些无聊的文本
#### 我是小学生，随便做的啦
--------------------------
### 1.输出进度条
### damn.loading(time，text，endtext)
### 指运行time秒，不断叠加text，运行完输出endtext。
--------------------------
### 2.转圈
### damn.turn(time,endtext)
### 指运行time秒，输出一根线不断旋转，运行完输出endtext。
--------------------------
### 3.输出
### damn.fo()
#### 指输出一个佛祖
